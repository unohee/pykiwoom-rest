# PRD: PyKiwoom-REST v2.2.0 - WebSocket 실시간 시세 조회

## 📋 개요
- **목적**: Kiwoom증권 REST API에 WebSocket 기반 실시간 시세 조회 기능 추가
- **범위**: 실시간 호가, 체결가, 시세 등 WebSocket API 구현
- **제약사항**:
  - Kiwoom API WebSocket 엔드포인트 사용
  - 기존 Rate Limiting 정책 유지 (초당 20회)
  - 토큰 기반 인증 (24시간 유효)
  - 최소 Python 3.8 호환성

## 📐 기술 명세

### WebSocket 엔드포인트
- **Base URL**: `wss://openapi.koreainvestment.com:9443/`
- **인증**: HTTP Authorization 헤더 (Bearer 토큰)
- **프로토콜**: JSON 기반 메시지 (Kiwoom 표준 포맷)

### 주요 TR 코드
- `H0STASP0`: 실시간 호가 (매수/매도 10단계)
- `H0STCNT0`: 실시간 체결 (가격, 거래량)
- `H0STCNI0`: 실시간 현재가 (시가, 고가, 저가, 종가)

### 메시지 포맷 (예시)
```json
{
  "header": {
    "tr_id": "H0STASP0",
    "tr_key": "005930"
  },
  "body": {
    "output": {
      "askp1": "70000",
      "bidp1": "69900",
      ...
    }
  }
}
```
