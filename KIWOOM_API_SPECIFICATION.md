## 주요 특징

1. **REST API 기반**: HTTP 요청/응답 구조
2. **인증**: OAuth2 토큰 기반 인증
3. **Rate Limiting**: 초당 20회 제한
4. **데이터 형식**: JSON
5. **실시간**: WebSocket 지원

## 참고사항

- 일부 API는 장시간에만 정상 작동
- 토큰 유효기간: 24시간 (자동 갱신 필요)
- 대량 데이터 조회 시 페이지네이션 지원
- 모든 날짜 형식: YYYYMMDD
- 시간 형식: HHMMSS (6자리)
