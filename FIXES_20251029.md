## API 응답 키 참고

### ka10005 (주식일주월시분요청)
- 엔드포인트: `/api/dostk/mrkcond`
- 요청: `{"stk_cd": "005930"}`
- 응답 키: `stk_ddwkmm` (목록)
- 필드:
  - `date`: 날짜 (YYYYMMDD)
  - `open_pric`: 시가
  - `high_pric`: 고가
  - `low_pric`: 저가
  - `close_pric`: 종가
  - `trde_qty`: 거래량
  - `for_poss`: 외인보유
  - `for_wght`: 외인비중

### ka10008 (주식외국인종목별매매동향)
- 엔드포인트: `/api/dostk/frgnistt`
- 요청: `{"stk_cd": "005930"}`
- 응답 키: `stk_frgnr` (목록)
- 필드:
  - `dt`: 일자
  - `wght`: 비중 (%)
  - `trde_qty`: 거래량
  - `chg_qty`: 변동수량 (순매수량) ⭐
  - `poss_stkcnt`: 보유주식수
