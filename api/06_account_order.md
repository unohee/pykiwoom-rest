## 주요 API 상세

### 계좌평가현황요청 (kt00004)
- **URL**: `/api/dostk/acnt`
- **설명**: 계좌 평가금액, 손익률 등 조회

### 주식 매수/매도주문 (kt10000/kt10001)
- **URL**: `/api/dostk/ordr`
- **설명**: 일반 주식 매수/매도 주문 실행
- **주요 파라미터**: 종목코드, 주문수량, 주문가격, 주문구분
