## 주요 API 상세

### 주식호가요청 (ka10004)
- **URL**: `/api/dostk/mrkcond`
- **설명**: 특정 종목의 실시간 호가 정보 조회
- **주요 응답**: 매도/매수 호가 10단계, 잔량 정보

### 체결정보요청 (ka10003)
- **URL**: `/api/dostk/stkinfo`
- **설명**: 실시간 체결 내역 조회
- **주요 응답**: 체결가, 체결량, 체결시간
