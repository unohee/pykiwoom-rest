# 실시간시세 API (WebSocket)

WebSocket을 통해 실시간으로 시세 데이터를 수신하는 API입니다.

## API 목록

| 코드 | API명 | 데이터 유형 |
|------|-------|------------|
| 00 | 주문체결 | 체결 |
| 04 | 잔고 | 잔고 |
| 0A | 주식기세 | 기타 |
| 0B | 주식체결 | 체결 |
| 0C | 주식우선호가 | 호가 |
| 0D | 주식호가잔량 | 호가 |
| 0E | 주식시간외호가 | 호가 |
| 0F | 주식당일거래원 | 기타 |
| 0G | ETF NAV | ETF |
| 0H | 주식예상체결 | 체결 |
| 0J | 업종지수 | 업종 |
| 0U | 업종등락 | 업종 |
| 0g | 주식종목정보 | 기타 |
| 0m | ELW 이론가 | ELW |
| 0s | 장시작시간 | 기타 |
| 0u | ELW 지표 | ELW |

| 1h | VI발동/해제 | 기타 |

## WebSocket 연결

- **URL**: `/api/dostk/websocket`
- **Protocol**: `wss://`
- **인증**: Bearer Token 필수

## 주요 실시간 데이터

### 주식체결 (0B)
- 실시간 체결 데이터 스트리밍
- 체결가, 체결량, 체결시간 포함

### 주식호가잔량 (0D)
- 실시간 호가 10단계 데이터
- 매도/매수 호가 및 잔량

### 주문체결 (00)
- 본인 주문 체결 알림
- 주문번호, 체결가, 체결수량
